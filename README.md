# C-Grimoire

Este repositório é uma coleção de projetos e implementações de estruturas de dados e algoritmos em C. Cada subdiretório contém um projeto distinto com seu próprio conjunto de arquivos de implementação e, em alguns casos, materiais de referência e testes.

## Conteúdo do Repositório

### 1. `Circular_queue/`

Este diretório contém a implementação de uma fila circular em C.
- **`queue.c` / `queue.h`**: Implementação da estrutura de dados de fila circular.

### 2. `List/`

Este projeto é uma implementação de uma lista encadeada em C.
- **`list.c` / `list.h`**: Implementação da lista encadeada.
- **`reference/`**: Contém arquivos de referência, incluindo outra implementação, um `Makefile` e testes.

### 3. `Sorting_algorithms/`

Este diretório contém uma biblioteca com vários algoritmos de ordenação.
- **`sorting_lib.c` / `sorting_lib.h`**: Biblioteca com as implementações dos algoritmos de ordenação.
- **`reference/`**: Contém materiais de referência, incluindo um programa para análise de desempenho (`analysis.c`) e um `Makefile`.

### 4. `Stack/`

Este diretório contém uma implementação de uma pilha (stack) em C.
- **`stack.c` / `stack.h`**: Implementação da estrutura de dados de pilha.
- **`reference/`**: Contém um exemplo de uso (`main.c`), um `Makefile` e um conjunto de testes automatizados para validar a implementação.
