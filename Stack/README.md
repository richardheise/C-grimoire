# Stack (Pilha)

Esta pasta contém a implementação de um tipo de dado abstrato de pilha genérica em C.

- `stack.h`: Arquivo de cabeçalho com a definição da estrutura da pilha e os protótipos das funções.
- `stack.c`: Implementação das funções da pilha.

Esta implementação de pilha é genérica e pode armazenar elementos de qualquer tipo, utilizando `void*`.
